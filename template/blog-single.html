IPL final score prediction

Introduction
Cricket has evolved over the years, with technology playing a significant role in transforming the game both on and off the field. One area where technology has made substantial advancements is in cricket analytics. With the rise of machine learning and data science, predicting match outcomes and individual player performances has become more accurate and reliable. In this blog essay, we explore the exciting world of IPL final score prediction using the PyCaret module, and how it is revolutionizing cricket analytics.

PyCaret is an open-source Python module that simplifies the end-to-end machine learning workflow. It provides a high-level interface and automates various stages of ML model building, including data preprocessing, feature selection, model training, hyperparameter tuning, and model evaluation. PyCaret's easy-to-use and intuitive nature make it an ideal choice for predicting IPL final scores accurately.

After data is preprocessed and trained the resluted model is stored in Google Cloud since we can large files esily in Git. After that model is retreived and used for predictions

The whole repo is containerized and pushed to docker repository and then to Google's Artifacct repository and deployed using GKE autopiolet
